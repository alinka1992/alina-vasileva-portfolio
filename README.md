# Портфолио Алины Васильевой

Интерактивное executive-портфолио коммерческого директора (CCO) и Head of Business Development.

Сайт автоматически публикуется в GitHub Pages после обновления ветки `main`.

## Локальный запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
